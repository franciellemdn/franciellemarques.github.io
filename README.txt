README — Francielle Marques Personal Academic Website
=======================================================

OVERVIEW
--------
This is the source file for the personal academic website of Francielle Marques
do Nascimento, AI researcher at the TIDE Research Group, Universitat Pompeu Fabra
(Barcelona, Spain).

The site is a single-file static website (index.html) with no external dependencies
beyond Google Fonts, requiring no build tools, frameworks, or server configuration.


FILE
----
index.html        Main and only source file. Contains all HTML, CSS, and JavaScript.


SECTIONS
--------
- Hero            Name, title, research tags, and links to Scholar, UPF profile, LinkedIn.
- About           Short biography, position, institution, location, and citation count.
- Projects        Selected initiatives: AI For Social Good, World Dream Project, UPF TIDE.
- Publications    5 most-cited papers + doctoral thesis, ordered by year (newest first).
- News            Press coverage and institutional announcements.
- Outreach        YouTube talks, podcasts, and public science events.
- Contact         Email and profile links (no form).

DEPLOYMENT
----------
No build step required. Simply place index.html (and francielle.png if available)
in any static hosting environment (GitHub Pages, Netlify, Vercel, etc.) or open
directly in a browser.

The profile photo is expected at: francielle.png (same directory as index.html).
If the image is missing the avatar area will display as a plain background.


FONTS
-----
Loaded via Google Fonts CDN (requires internet connection to render correctly):
- Lora (serif) — headings and titles
- Inter (sans-serif) — body text and UI elements
